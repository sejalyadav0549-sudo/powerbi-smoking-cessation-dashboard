# 🚭 Smoking Cessation Dashboard (Power BI)

## 📌 Project Overview
This project presents an interactive dashboard built using Power BI to analyze smoking behavior and cessation patterns. It helps in understanding trends and supporting health-related insights.

## 🎯 Objectives
- Analyze smoking habits
- Track quit attempts
- Study demographic trends
- Provide actionable insights

## 🛠️ Tools Used
- Power BI  
- Power Query  
- DAX  
- Data Visualization  

## 📊 Dashboard Features
- Smokers vs Non-smokers analysis  
- Quit attempts tracking  
- Age & gender distribution  
- Smoking frequency insights  
- Interactive slicers  

## 📈 Key Insights
- Certain age groups show higher smoking rates  
- Quit attempts vary across demographics  
- Behavioral trends identified through visualization  

## 🚀 Conclusion
This dashboard demonstrates data analysis and visualization skills using Power BI.
